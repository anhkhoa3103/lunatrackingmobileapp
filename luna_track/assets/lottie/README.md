# Lottie animations

Download the following free Lottie JSON files into this folder:

- `empty_chart.json` — from https://lottiefiles.com/animations/no-data
  (search "empty chart" or "no data" — pick one with a pink/colorful theme)
- `loading.json` — optional loading animation

Until `empty_chart.json` is added, the insights empty state falls back
to a 📊 emoji via the Lottie `errorBuilder`.
